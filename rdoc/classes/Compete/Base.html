parse(xml)

Used to parse the XML response into a Compete specific hash, if one would like. Retreives the latest stats available from the Compete XML response

Example

  @compete_info = @compete.parse(compete_xml)

Result

  {:rank=>"1", :uv_ranking=>"1", :uv_count=>"135,834,459", :month=>"06", :year=>"2008", :domain=>"google.com"}
